# Mini RPG Art Bible v1.0

## コンセプト

「朝の海辺を歩きたくなるRPG」

静かで優しい世界。
海風が吹き、木漏れ日が揺れ、
犬がいて、人々は穏やかに暮らしている。

主人公は世界に馴染むが、
背景に埋もれない存在感を持つ。

## Master Sprite

主人公スプライト制作における、**最優先のビジュアル基準**。

### 基準ファイル

`sprites/player/player_master_v1.png`

AI および制作者は、文章による説明よりも **Master Sprite の見た目を優先** して参照する。

### 統一する要素

Master Sprite と以下を統一する。

| 要素 | 内容 |
|------|------|
| 頭身 | 約4頭身 |
| 配色 | 赤髪・青い服・緑の差し色（主人公は世界よりやや高彩度） |
| シルエット | Master Sprite の全体バランス・読みやすさ |
| 輪郭 | 濃い茶色のアウトライン（ドットルール準拠） |
| 服装 | 青い上着、緑のアクセント、海辺の町に似合う自然な装い |
| 髪型 | 鮮やかな赤髪（主人公のトレードマーク） |
| 小物 | バッグなど、Master Sprite で定義された付属品 |

### 適用範囲

以下を制作・更新するときも、Master Sprite を基準とする。

・待機ポーズ
・歩行アニメーション
・向き変更（上下左右）
・その他の主人公スプライト

### バージョン管理

Master Sprite は v2、v3… と更新される可能性がある。

**常に最新版を制作基準** とし、旧版を基準に新規 Asset を制作しない。

関連ドキュメント: [CHARACTER_BIBLE.md](./CHARACTER_BIBLE.md)（デザイン履歴・レビュー）
