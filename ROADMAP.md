## ファイル構成（現状）

```
index.html      … エントリーポイント
style.css       … スタイル（会話ウィンドウ含む）
input.js        … キー入力
map.js          … マップデータ・描画
npc.js          … NPC 配置・描画
dialogue.js     … 会話データ・ウィンドウ制御
interaction.js  … 前方判定・会話トリガー
collision.js    … 当たり判定
camera.js       … カメラスクロール
script.js       … ゲームループ・主人公
```
